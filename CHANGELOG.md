## [1.0.0] - 2025-12-26

### Added
- Initial release of Vroomfogle & Company website
- Homepage with parallax scrolling sections featuring Shadow World imagery
- "Who is Vroomfogle?" introduction section with character backstory
- Ask Andraax page: Single Q&A interface with cryptic oracle persona
- Visit Nomikos page: Interactive chat interface with conversation history
- 25 curated example queries for Shadow World lore exploration
- Markdown rendering support using marked.js library
- Era-aware responses with SE (Second Era) and TE (Third Era) distinctions
- Responsive design with purple/gold fantasy RPG theme
- Glassmorphism effects and smooth animations
- Integration with Nomikos API at nomikos-api.vroomfogle.com
- GitHub Pages deployment configuration with custom domain support
- Comprehensive README with deployment instructions
- Example images: castle, library, characters, and parallax backgrounds

### Technical
- Pure HTML/CSS/JavaScript architecture (no build process required)
- CSS custom properties for consistent theming
- Vanilla JavaScript for all interactions
- CDN-based marked.js for markdown parsing
- Mobile-first responsive design
- Accessibility considerations in HTML structure
